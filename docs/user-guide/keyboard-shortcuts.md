# Keyboard Shortcuts

This page lists all keyboard shortcuts available in the YouTube Research Video Clip Finder application.

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+/` or `Cmd+/` | Show keyboard shortcuts |
| `Ctrl+S` or `Cmd+S` | Save current project |
| `Ctrl+N` or `Cmd+N` | New project |
| `Ctrl+O` or `Cmd+O` | Open project |
| `Ctrl+F` or `Cmd+F` | Focus search bar |
| `Esc` | Close current modal or cancel operation |
| `F1` | Open help |

## Search Interface

| Shortcut | Action |
|----------|--------|
| `Enter` | Execute search |
| `Alt+Enter` | Open advanced search options |
| `Ctrl+Alt+F` or `Cmd+Alt+F` | Toggle filter panel |
| `Tab` / `Shift+Tab` | Navigate through search results |
| `Ctrl+1-9` or `Cmd+1-9` | Select search result by number |

## Video Player

| Shortcut | Action |
|----------|--------|
| `Space` | Play/Pause |
| `J` | Rewind 10 seconds |
| `K` | Play/Pause |
| `L` | Forward 10 seconds |
| `←` | Rewind 5 seconds |
| `→` | Forward 5 seconds |
| `↑` | Volume up |
| `↓` | Volume down |
| `M` | Mute/Unmute |
| `F` | Toggle fullscreen |
| `0-9` | Seek to 0-90% of video |
| `.` | Next frame (when paused) |
| `,` | Previous frame (when paused) |

## Clip Editor

| Shortcut | Action |
|----------|--------|
| `I` | Set in-point (start of clip) |
| `O` | Set out-point (end of clip) |
| `Ctrl+B` or `Cmd+B` | Create clip from current in/out points |
| `Delete` | Remove selected clip |
| `Ctrl+Z` or `Cmd+Z` | Undo last action |
| `Ctrl+Shift+Z` or `Cmd+Shift+Z` | Redo last action |
| `Ctrl+C` or `Cmd+C` | Copy selected clip metadata |
| `Ctrl+V` or `Cmd+V` | Paste clip metadata to selected clip |

## Metadata Editor

| Shortcut | Action |
|----------|--------|
| `Ctrl+T` or `Cmd+T` | Add new tag |
| `Alt+↑` | Move to previous metadata field |
| `Alt+↓` | Move to next metadata field |
| `Ctrl+Enter` or `Cmd+Enter` | Save metadata changes |

## Timeline Navigation

| Shortcut | Action |
|----------|--------|
| `Ctrl+Wheel` or `Cmd+Wheel` | Zoom timeline in/out |
| `Alt+Drag` | Pan timeline |
| `Home` | Go to beginning of timeline |
| `End` | Go to end of timeline |
| `Shift+Drag` | Select multiple clips |

## Accessibility Shortcuts

| Shortcut | Action |
|----------|--------|
| `Alt+1-9` | Navigate to main application sections |
| `Ctrl+Alt+Z` or `Cmd+Alt+Z` | Toggle high contrast mode |
| `Ctrl+Alt+X` or `Cmd+Alt+X` | Toggle larger text mode |
