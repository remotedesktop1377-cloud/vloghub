# Quick Reference Guide

## Basic Workflow

1. **Search**: Enter your research topic in the search bar
2. **Review**: Browse through search results and preview videos
3. **Extract**: Select relevant segments and trim clips
4. **Tag**: Add metadata to your clips
5. **Download**: Save clips to your preferred location

## Common Tasks

### Creating a New Project

1. Click "New Project" on the dashboard
2. Enter project name and description
3. Select storage location
4. Click "Create"

### Searching for Videos

1. Enter search terms in the main search bar
2. Use filters to narrow results:
   - Date range
   - Duration
   - Channel
   - Caption availability
   - License type
3. Click on results to preview

### Extracting Clips

1. Play the video and identify relevant segment
2. Press `I` to mark in-point (start)
3. Press `O` to mark out-point (end)
4. Press `Ctrl+B` or `Cmd+B` to create clip
5. Adjust clip boundaries by dragging handles in timeline

### Adding Metadata

1. Select clip in timeline
2. In Metadata panel:
   - Add title
   - Set date
   - Select location
   - Choose speaker
   - Add tags
3. Press `Ctrl+Enter` or `Cmd+Enter` to save

### Downloading Clips

1. Select clips in timeline
2. Click "Download" button
3. Choose format and quality
4. Select destination folder
5. Click "Start Download"

## Troubleshooting

### Search Not Working
- Verify YouTube API key in Settings
- Check internet connection
- Try different search terms

### Video Won't Play
- Refresh the page
- Try a different browser
- Check if video is available in your region

### Download Failed
- Ensure sufficient disk space
- Check write permissions for destination folder
- Try a different format
