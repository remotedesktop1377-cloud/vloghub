# Interactive Tutorials

This directory contains interactive tutorials to help you learn how to use the YouTube Research Video Clip Finder effectively.

## Available Tutorials

1. [Getting Started Tutorial](./getting-started-tutorial.md)
   - Learn the basics of the application
   - Create your first project
   - Perform your first search

2. [Advanced Search Techniques](./advanced-search-tutorial.md)
   - Learn how to use filters effectively
   - Understand query expansion
   - Use date and region-specific searches

3. [Clip Extraction Masterclass](./clip-extraction-tutorial.md)
   - Identify relevant segments quickly
   - Use keyboard shortcuts for efficient trimming
   - Work with the timeline editor

4. [Metadata Management](./metadata-tutorial.md)
   - Add comprehensive metadata to clips
   - Use automatic tagging features
   - Organize clips with custom taxonomies

5. [Project Organization](./project-organization-tutorial.md)
   - Set up folder structures
   - Create naming conventions
   - Manage multiple research projects

## How to Use These Tutorials

Each tutorial is designed to be interactive, with step-by-step instructions and exercises. To get the most out of these tutorials:

1. Follow along in the actual application
2. Complete each exercise before moving to the next
3. Use the provided sample projects when available

## Video Walkthroughs

For visual learners, we also provide video walkthroughs for each tutorial. These can be found in the [videos](../videos/) directory.
